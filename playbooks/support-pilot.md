# Support Pilot Playbook

## Recommended starting workflow

Tier 1 reply drafting and knowledge retrieval for repeatable case types.

## Suggested metrics

- agent adoption in target queues
- average handle time
- first-contact resolution
- escalation rate
- draft acceptance rate
- policy exception rate

## Recommended systems

- Zendesk or Salesforce Service Cloud
- knowledge base platform
- QA review workflow
