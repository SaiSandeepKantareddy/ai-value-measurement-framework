# 90-Day Rollout Playbook

## Phase 1: Prove

- run one or two pilots
- standardize metric definitions
- validate instrumentation quality

## Phase 2: Expand

- extend to adjacent workflows
- add executive reporting
- document decision rules for scale and pause actions

## Phase 3: Operationalize

- establish monthly and quarterly review cadences
- map governance controls to workflow criticality
- create a reusable integration pattern for source systems

## Success criteria

- repeatable scorecards across functions
- common event definitions
- clear operating cadence for leaders
- evidence-based prioritization of AI investments
