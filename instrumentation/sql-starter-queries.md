# SQL Starter Queries

These are conceptual examples showing how teams might derive scorecard metrics once events are collected.

## Adoption example

Count unique eligible users with at least one accepted AI-assisted workflow event in the last 7 days.

## Speed example

Measure median time between workflow start and workflow completed events for assisted versus unassisted work.

## Override example

Calculate override rate as workflows with `human_override_recorded` divided by workflows with `ai_suggestion_accepted`.
