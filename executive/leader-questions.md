# Leader Questions

These are the questions a serious leader should ask in every AI value review.

## Business value

- Which workflows created measurable value this period?
- Which high-adoption workflows still have weak business impact?
- Which teams are ready to scale?

## Measurement quality

- Which metrics do we trust most?
- Where are instrumentation gaps affecting interpretation?
- Which scorecards are still too immature for executive decisions?

## Governance

- Where are override rates highest?
- Which workflows are closest to control failure?
- Are review requirements matched to workflow risk?

## Decision discipline

- What should we stop funding?
- What should we scale next?
- What should we fix before expanding?
